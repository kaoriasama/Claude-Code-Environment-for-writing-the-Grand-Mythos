# 社会構造

このファイルは作品世界に存在する構造的な位置（ポジション）を定義する。
**ここに定義されるのは人物ではなく、人物が配置されるべき位置である。**

## 記述方針

- 位置には利害・制約・視野を持たせる。これにより、そこに配置された人物が何を見て何を求めるかが自動的に決まる。
- すべての位置が埋まっている必要はない。空席は空席として明示する。
- 位置は固定ではない。人物が構造内で移動することが物語の駆動力になる場合がある。
