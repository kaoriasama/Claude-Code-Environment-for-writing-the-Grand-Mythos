# Change Proposals

このファイルは Claude Code からの正典（Input/）への変更提案を蓄積するログである。

## 使用規則

1. このファイルは**追記専用**とする。過去のエントリを遡って編集しない。
2. 提案は対話順に上から蓄積する。
3. 人間が上から順に実行すれば、すべての操作が完了できる状態を保つ。
4. 後の提案は、先の提案が実行済みであることを前提に記述する。
5. ユーザーが提案を実行し終えた場合、実行済みのエントリに `[実行済み]` マークを付与してもよい。

## エントリの形式

各提案は以下のテンプレートに従う。

```
---
### [提案番号]: [一行要約]

対象ファイル: （絶対パス、または新規作成時の予定パス）
変更方法: 追記 / 挿入 / 書換 / 削除 / 移動 / 新規 / 分割
変更箇所: （セクション名・見出し・行番号など、人間が特定できる粒度で）
変更内容:
（具体的なテキスト。削除の場合は ~~取り消し線~~ で明示）
変更理由: （省略可。正典の修正提案時は必須）
---
```

## 変更方法の定義

- **追記**: セクション末尾または文末への付加
- **挿入**: 指定箇所への割り込み
- **書換**: 既存テキストの置換
- **削除**: 対象を ~~取り消し線~~ で明示
- **移動**: 移動元（削除エントリ）と移動先（挿入エントリ）を別エントリで記述
- **新規**: ファイル新規作成。変更箇所欄にパスを記載
- **分割**: 削除エントリと新規エントリの組み合わせで記述
